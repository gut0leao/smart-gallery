   - Upload the plugin files to `/wp-content/plugins/smart-gallery-filter/`

== Frequently Asked Questions ==

= Do I need Elementor Pro? =

No! Smart Gallery Filter works with the free version of Elementor. It's designed as a free alternative to Elementor Pro's Posts widget.

= Do I need Pods Framework? =

Yes, Pods Framework is required. The plugin is specifically designed to work with Custom Post Types and custom fields created by Pods Framework.

= What post types are supported? =

Custom Post Types created with Pods Framework, including:
- Any CPT created through Pods admin interface
- Built-in post types extended with Pods custom fields

= Can I customize the appearance? =

Yes! The plugin includes:
- Multiple layout options (sidebar positions)
- Responsive grid settings
- Custom message styling
- CSS classes for advanced customization

= Does it work on mobile? =

Absolutely! The plugin is mobile-first with responsive breakpoints:
- Large screens: Up to 6 columns
- Tablets: 2 columns
- Mobile: 1 column
